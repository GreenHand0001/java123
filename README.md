# 超市管理系统 java123

## 项目概述

本项目是基于B/S架构的超市管理系统，采用MVC模式设计，实现了前后端分离的开发模式。系统后端采用Java语言，基于SpringBoot框架，前端采用Vue技术构建用户界面。

## 技术栈

- 后端：Java、SpringBoot、Mybatis-Plus、Redis
- 数据库：Mysql
- 前端：Vue
- 开发环境：IDEA、JDK1.8、Maven

## 系统角色

- 管理员：负责系统管理、角色权限分配、销售管理、库存管理、人事管理等功能模块的操作。
- 用户：可进行商品浏览、购物车管理、订单查看、个人中心管理等操作。

## 功能模块

### 管理员模块

- 首页：统计图表，存库预警
- 销售管理：销售主页，销售记录，积分兑换记录
- 人事管理：部门管理，员工管理
- 库存管理：仓库管理，出库明细，入库明细，供货商信息，库存统计
- 用户管理：用户列表，设置会员
- 公告管理：公告列表
- 订单管理：订单列表，订单评价列表
- 系统管理：角色管理，菜单管理
- 商品管理：分类管理，商品信息，积分商品，销售统计

### 用户模块

- 首页
- 公告
- 商品浏览
- 购物车
- 我的订单
- 个人中心

- 灵活的权限管理：管理员可以根据需要自由设置角色并分配权限。
